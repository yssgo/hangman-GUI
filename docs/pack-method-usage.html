위젯

  • 위젯 ( padx, pady, ipadx, ipady 의 기본값은 0)

      +--------------------------------------+
                   pady
  .      +------------------------+      .
         |        ipady           |
  .      |       + + + + +        |      .
    padx | ipadx + 내용  + ipdadx | padx
  .      |       + + + + +        |      .
         |         ipady          |
  .      +------------------------+      .
                   pady
  +--------------------------------------+

꾸러미

  • 꾸러미 ( side='top 또는 side='bottom' 일 때 )

      +--------------+-----------------------------+----------------+
  | 왼쪽 빈 공간 |             위젯            | 오른쪽 빈 공간 |
  +--------------+-----------------------------+----------------+

  • 꾸러미 ( side='left 또는 side='right' 일 때)

      +-----------+
  |   위쪽    |
  |  빈 공간  |
  |           |
  +  - - - -  +
  |           |
  |    위     |
  |           |
  |    젯     |
  |           |
  +  - - - -  +
  |           |
  |  아래쪽   |
  | 빈 공간   |
  +-----------+
